Exercise 1: CSS Selector

Use CSS selectors and the color-related property to style each box accordingly.

Your goal is to write CSS rules in the <style> tag that will color each box according to its label. For example, if a box says "div (element) - color: red", you should write a CSS rule using the p element selector to implement the changes.

Tasks: Try to follow the constrains given in each task. Don't change the HTML structure, only add CSS rules to achieve the desired styling.
